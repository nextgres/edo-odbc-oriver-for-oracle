Welcome to the ODBC SQL Reference guide

This guide describes a non existent form of SQL (Standard Query Language) that does not exist anywhere in the world. So why this guide? What it does describe is the intermediate forms of SQL that can be used / expressed by using a standard ODBC interface to an existing RDBMS (Relational DataBase Management System). In case of doubt: consult the documentation of your RDBMS!

The ODBC (Open DataBase Connectivity) standard makes it possible to manage, use and support databases of a various vendors, years and depth to be handled in the same standard way.

This SQL Reference is not the complete documentation for your SQL language. For more detailed information see the documentation of your RDBMS.
