REVOKE (System Privileges)

Use this command to revoke system privileges and roles from users and roles. To revoke object privileges from users and roles, use the REVOKE (Object privileges)

Syntax of the SQL statement

REVOKE USAGE

ON system-object FROM user-name [,...]

[ CASCADE | RESTRICT ]

;

system-object ::=

 

  CHARACTER SET

| COLLATION

| DOMAIN

| TRANSLATION

user-name ::=

 

letter { letter | digit | '_' [, ...] }

 

Too see which options of the revoke statement are supported, look up the grant statement in the ODBC Info tree of the Open ODBCQueryTool under 'SQL Supported standard / Revoke'.

See also: GRANT (System Privileges and Roles) , REVOKE (Schema Object Privileges)
