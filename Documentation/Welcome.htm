Welcome to EDO

EDO is the Open Source Open DataBase Connectivity (ODBC) driver for the Oracle databases 9i, 10i and 11g.

This guide describes the EDO Oracle driver in five parts:

    1. Installation: Installing the EDO driver set on your computer

    2. Administration of datasources: How to create and modify datasources

    3. ODBC SQL Quick reference: The greatest common denominator of SQL grammar

    4. EDO for programmers: Special information for ODBC programmers

    5. Differences with Oracle ODBC: Where EDO differs from the official Oracle ODBC driver
