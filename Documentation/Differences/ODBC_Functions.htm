ODBC Functions

The ODBC "{ fn"  function escape syntax makes it possible to use one set of functions for all database accessed by a ODBC application. A programmer need not check with the local database manual to use these functions. Below is a table of all functions in the ODBC standard. As you can see, EDO supports all these functions.
