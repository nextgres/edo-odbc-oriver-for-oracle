Datatypes discovery

Which datatypes a database will support can be retrieved by way of the "SQLGetTypeInfo" ODBC function. Strict portable applications will never attempt to use datatypes not reported back by this function. So in order to allow for maximal portability an ODBC driver should report as much datatypes back to the application as it can.
