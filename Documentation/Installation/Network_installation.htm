Network installation

You can install the EDO driver in silent mode with the "/S" option to the executable installation program. This enables the program to be installed as part of login scripts on networks or be pushed from the Active Directory as part of the organisations policy. The /S option is case sensitive, so /s will not work.

But beware: The user running the installer in this mode must at least have local administration rights to do so.

Example:

Put the following line in your login script:

X:\Path_to_the_installer_program\EDO_Installer_1.0.0.exe /S
