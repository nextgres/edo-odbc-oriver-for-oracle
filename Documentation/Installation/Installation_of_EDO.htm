Installation of EDO

To install the EDO Oracle ODBC driver on your system, you need to download the installer file first. Normally this is the file named "EDO_Installer_x.y.z.exe", where x.y.z stands for the version number. So for version 1.0.0 this would be the file named "EDO_Installer_1.0.0.exe".

1) After you double-click on the file you might see this warning if you did have it installed previously. Installing a newer version will of course not trigger this warning.

2) After this warning (if you continue) you will see the EDO logo with the image of the castle of EDO made out of Lego's. O, by the way "Ja" is dutch for "Yes" and "Nee" means "No".

3) After the logo has finished display-ing you will see the welcome screen of the installer. This is just a welcome screen with text to tell you that you can install the EDO Oracle ODBC driver by means of it's wizard.

4) After you push the "Next >" button, you will see the GNU General public license. Read this carefully and only proceed with the "I Agree" button if you think you can live by the standards this license sets.

5) After you have agreed to the license, you will see the directory screen. Here you can choose where you want to install your EDO driver. As a default, it will prompt you to install it in the "Program Files\EDO" directory. As there are no log files or otherwise writable files in EDO, this is a good place, even for a MS-Vista system with Users Access control.

6) One thing left to do: Push the "Install" button to install it. The installation rushes by and that leaves you with the final success page and a finish button. Pushing it will close the wizard. Your done. The EDO driver has been installed.

7) If you don't want to go through the whole setup process by hand (and mouse!), you can also do a silent network installation. This type of installation is especially suited for scripting.
