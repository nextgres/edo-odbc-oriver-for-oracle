Creating a datasource

Once you have installed the EDO ODBC driver, creating a datasource is a breeze. Just begin by starting the ODBC-manager from your start menu ("Settings / System administration / Datasources (ODBC)").

Now you can create a user datasource or a system datasource by choosing the first (users) or second (system) tab from this manager and then the "Add..." button. Now you can choose the "EDO Oracle Driver" and can push the "Complete" button.

Once you have done so, you will enter the EDO setup wizard:

Here you must complete a few mandatory entries and a some optional ones. These entries are described on the following pages:

- Defining the datasource

- ODBC Settings of the datasource

- Oracle settings of the datasource

- Tracing and translation

- National Language Support settings
