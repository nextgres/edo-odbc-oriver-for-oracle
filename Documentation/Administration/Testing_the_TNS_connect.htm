Testing the TNS connect string

If you are unsure whether the "TNS connectstring" will work for your database, you can test so by using the oracle "tnsping" utility. This utility is like a internet ping utility, but specially for an oracle database. Testing involves the following steps:

1) Start a command shell by using the start button and choose "Run..."

2) Type in the command line "cmd" and press enter;

3) Type in the tnsping command and the name of your TNS connect string. In this example the name of the database is "magenta7"

After pressing enter, you will see something similar to this output. Here it has found the database with a roundtrip time of just 20 milliseconds.

If your TNS connect string doesn't work, it will take slightly longer for the tnsping utility to answer. It will then do so with a TNS error code: TNS-03505: Name not found.
