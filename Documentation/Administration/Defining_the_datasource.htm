Defining the datasource

The first step in defining a datasource is by specifying three things in the top part of the wizard:

1) The name of the datasource (mandatory)

2) A description of the datasource (optional)

3) The TNS connect string for the Oracle SQL*NET layer (mandatory).

After specifying the names in step 1 and 3, you can already test your connection with the "Test connection" button. You can do so by trying the infamous scott/tiger combination if you activated it in the installation process of your database. Otherwise you will have to use any other user/password combination available to you.
