Tracing and translation

The third tab with information about the ODBC datasource is called "Tracing & Translation". This tab contains two file oriented options:

 

Field name

Description

Use tracing

Select this field if you want to use tracing of the ODBC calls. Use with caution and for debugging purposes only, as it takes LOTS of performance.

Tracefile

Specify the file with your trace info, or

Trace button [...]

Use this button to select a trace file for your tracing information.

Translation option

Standard numeric option (unsigned 32 bit number) to be givven to each call of the translation library. This value depends on the translation library you are using.

Translation library

This is the DLL containing the SQLDriverToDatasource and the SQLDatasourceToDriver functions where you can override the codepage translations of strings.

Translation button [...]

Use this button to select a translation *.DLL file.
