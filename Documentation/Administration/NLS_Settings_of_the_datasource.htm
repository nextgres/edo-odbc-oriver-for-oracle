NLS Settings of the datasource

The fourth tab with information about the ODBC datasource is called "NLS". This tab contains the National Language Support (NLS) settings.

 

Field name

Description

National language support

Use the combobox to choose one out of three options:

1) Use standard Oracle NLS settings. (use the login button to see them)

2) Use standard ODBC NLS settings. (displayed directly in the four fields below)

3) Use specific Oracle NLS settings. Set them in the four fields below

Session login button

Use this button in case you use the standard Oracle NLS settings. They will be read from the database connection.

Formatting of a date

Standard Oracle format for a date, as used in the TO_DATE function.

Formatting of time

Standard Oracle time format as used in the TO_DATE or TO_TIMESTAMP function

Formatting of timestamp

Standard Oracle timestamp format as used in the TO_TIMESTAMP function

Timestamp + timezone

Standard Oracle timestamp with timezone format as used in the TO_TIMESTAMP function with datatypes that do support the timezone.

 

N.B.1: The NLS values shown here are European values (The Netherlands to be precise).

N.B.2: The four fields at the right of the value fields are for future expansion. The have no purpose now.
